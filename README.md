## Usage

Clone the repository:
```bash
git clone https://github.com/loljoho/my-first-calculator.git
```

Enter the directory:
```bash
cd my-first-calculator/
```

Install stuff:
```bash
npm install
```

Run it:
```bash
npm start
```
